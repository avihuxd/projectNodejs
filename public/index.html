Toys API Documentation!

  • Search

    Use the following URL:
    http://Domain/toys/search/?s=
    use this get request to search a toy/s by the name or information

  • Price range

    use this request to get toys in between minimum and maximum range price you inserted (replace your values where the ** are displayed):
    Use the following URL:
    http://Domain/toys/prices/?min=*MININUM-VALUE*&max=*MAXIMUM-VALUE*

  • By page

    use this request to get toys in your requested page you inserted: (replace your page number where the ** are displayed)
    Use the following URL:
    http://Domain/toys/?page=*NUMBER*

  • By category

    Use this request to search a category of toys you want: (replace the category you are looking for where the ** are displayed)
    Use the following URL:
    http://Domain/toys/cat/*CATEGORY*

Add a new product - must register first and login and have a token to send this request. (POST)
Use the following URL:
http://Domain/toys
  • name -Mandatory, must be filled with a string
  • info -Mandatory, must be filled with a string
  • category -Mandatory, must be filled with a string
  • img_url -Can be sent empty, or be filled with a string.
  • price -Mandatory, must be filled with a number

Here is an example of how a post request of new toy looks like(Must have a token):

{
"name":"Crayola Color Wonder Magic Light Brush, Mess Free Painting, Gift for Kids, 3, 4, 5, 6",
"info":"Painting is magical with the Color Wonder Magic Light Brush Though Color Wonder Paints are clear, the brush lights up to indicate the color Color Wonder Paints are mess-free and only appear on special Color Wonder Paper, not on skin, clothing, or walls. So kids can explore the joy of painting and moms can let them create without worry Makes a great holiday or birthday gift for kids ages 3 and up (Clearance)",
"category":"Storytelling",
"price":29.99,
"img_url":"https://slimages.macysassets.com/is/image/MCY/products/1/optimized/18288871_fpx.tif"
}

Here you can send a post request to sign up for the website.
Use the following URL:
http://Domain/users/signup
You must send mandatory 3 parameters:
  1. Name: "String" *required*
  2. Email: "String" *required*
  3. Password: "String" *required*

Use this post request to get an access for adding a new toy, and to edit or delete a toy you inserted.
Use the following URL:
http://Domain/users/login
Instructions: You must send mandatory 2 parameters:
  1. Email: "String" *required*
  2. Password: "String" *required*
